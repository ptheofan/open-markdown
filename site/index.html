Open Markdown

A beautiful, native macOS app for reading Markdown files — with GitHub-style rendering, diagrams, and live reload.

Download Latest Release View on GitHub

Features

GitHub Flavored Markdown

Full GFM support including tables, task lists, strikethrough, and autolinks — rendered exactly like GitHub.

Syntax Highlighting

Beautiful code blocks with automatic language detection powered by highlight.js, covering 190+ languages.

Mermaid Diagrams

Render flowcharts, sequence diagrams, Gantt charts, and more directly from fenced code blocks.

Live Reload

Automatically re-renders when the file changes on disk, with change highlighting so you can see exactly what changed. Edit in your favourite editor, preview instantly.

Multi-Window

Open multiple files side by side with Cmd+N. Each window tracks its own file independently.

Find in Page

Search within rendered content using Cmd+F. Quickly locate text in long documents.

Export as Image

Save the entire document as a PNG image, or right-click any Mermaid diagram to copy it as an image.

Copy as Rich Text

Copy the rendered document to your clipboard as rich text. Paste directly into Google Docs, Jira, Confluence, and more — formatting preserved.

Light & Dark Themes

Follows your system appearance or toggle manually. Looks great in both modes.

Drag & Drop

Drop any Markdown file onto the app window or Dock icon to open it immediately.

Frequently Asked Questions

How do I set Markdown Viewer as the default app for .md files?

Right-click any .md file in Finder, select Get Info, choose Markdown Viewer under Open with, then click Change All.... See the documentation for details.

Does the app auto-update?

Not currently. Download the latest version from the GitHub Releases page.

Does Markdown Viewer send any data over the network?

No. The app operates entirely offline. No analytics, telemetry, or network requests of any kind. See the privacy policy for details.

Which macOS versions are supported?

Markdown Viewer supports macOS 13 (Ventura) and later.

Can I use it to edit Markdown?

Markdown Viewer is a read-only viewer. For editing, use your favourite text editor — the app will live-reload the preview whenever you save.

My Mermaid diagram isn't rendering. What should I do?

Ensure the code block uses the mermaid language tag. Check the Mermaid docs for supported syntax. If the diagram is valid but still not rendering, please open an issue.
